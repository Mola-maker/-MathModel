# MCM/ICM 论文格式规范

## 基本要求
- 摘要（Summary）：1页以内，独立成页
- 论文正文：不超过25页（含图表）
- 字体：12pt，Times New Roman 或类似字体
- 页边距：1英寸（2.54cm）
- 页眉：团队编号（每页）

## 章节结构（标准）
1. Summary Sheet（摘要页）
2. Introduction
3. Assumptions and Justifications
4. Model Development / Formulation
5. Solution and Results
6. Sensitivity Analysis
7. Strengths and Weaknesses
8. Conclusion
9. References

## 图表规范
- 图：Figure X. 标题（居中，图下方）
- 表：Table X. 标题（居中，表上方）
- 所有图表需在正文中引用

## 公式规范
- 所有公式编号：(1)(2)...
- 变量首次出现时定义
- 使用国际标准符号

## 摘要写作要求
- 问题重述（1-2句）
- 方法概述（核心模型）
- 主要结论（量化结果）
- 创新亮点
- 关键词（3-5个）
